Пример шифрования RSA

Открытый ключ:

n произведение двух простых чисел р и q (p и q должны храниться в секрете) , е число взаимно простое с (p-1)(q-1)

Закрытый ключ:

d = ei mod ((p-1)(q-1))

Шифрование:

сi = тiе mod n

Дешифрирование:

тi= cid mod n

Короткий пример пояснит работу алгоритма . Если р = 47 и q = 71, то

п=p*q = 3337

Ключ е не должен иметь общих множителей с числом (p-1)(q-1)

(p-1)(q-1)= 46*70 =3220

Выберем (случайно) е равным 79. В этом случае d = 79-1 mod 3220 =1019

При вычислении этого числа использован расширенный алгоритм Эвклида. Опубликуем е и n, сохранив в секрете d. Сохраним в секрете так же р и q. Для шифрования сообщения

m = 6882326879666683

сначала разделим его на маленькие блоки. Для нашего случая подойдут трехбуквенные блоки. Сообщение разбивается на шесть блоков mi:

m1= 688

m2 = 232

m3 = 687

m4 = 966

m5 = 668

m6 = 003

Первый блок шифруется как 68879 mod 3337 = 1570 = c1

Выполняя ту же операцию для последующих блоков, создаём шифротекст сообщения :

с = 1570 2756 2091 2276 2423 158

Для дешифрирование нужно выполнить возведение в степень, используя ключ дешифрирования 1019:

15701019 mod 3337 = 688 = m1

Аналогично восстанавливается оставшаяся часть сообщения.
